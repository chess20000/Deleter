Deleter

macOS 照片 / 视频筛选器。浏览、预览、标记丢弃，生成压缩副本后丢弃原片。

下载 Deleter.dmg GitHub

macOS 14+ · v1.0 · Apache-2.0

核心功能

Finder 风格网格

文件夹与媒体混合展示，等间距方形缩略图，按需懒加载生成。

快速预览

空格全屏 Quick Look，方向键切换，支持缩放与视频播放。

键盘驱动

方向键导航，D 标记丢弃，回车进入或预览，退格返回上层。

RAW + JPG 配对

自动识别同 base 名的 RAW/JPG，可合并或分开显示。

压缩副本 B-side

标记项在 _bside 子目录生成压缩副本：图片转 JPEG，视频重编码。

丢弃原片

副本生成完成后，一键将原片移入废纸篓，安全腾出空间。

安装

下载 dmg，将 Deleter.app 拖入「应用程序」。未签名，首次打开需移除隔离属性：

# 移除 macOS 隔离属性
xattr -dr com.apple.quarantine /Applications/Deleter.app
